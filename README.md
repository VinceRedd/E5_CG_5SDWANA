# Projet Kubernetes — PoC E-commerce

Ce projet est un **Proof of Concept Kubernetes (K8s)** réalisé dans le cadre d’un projet SRE.  
Il démontre le déploiement d’une architecture conteneurisée avec **3 applications Django**, incluant une application e-commerce intégrant **Stripe**.

## 🧱 Stack technique

- Kubernetes (Minikube)
- Docker (images multi-stage)
- Python / Django
- Apache Benchmark (test de charge)
- Stripe (paiement en mode test)
